About

Kitsuchan is here to make your guild more fun! Please check the support server for announcements about changes and new features. Kitsuchan's prefix is kit, or you can mention the bot. Run kit help for a full command listing, or alternatively check the GitHub page.

Features

  • Foxgirls
  • A couple different image searches
  • Meme generator
  • Reaction images (kiss, poke, hug, and more)
  • MyAnimeList lookup (anime, manga)
  • Wikipedia search
  • Color command (HLS, RGBA, hex codes, thumbnail)
  • Information commands - Who has what role? What custom emojis are there? Some others, too.
  • Random trivia
  • Tripmines to explode people!
  • Magic 8-ball
  • Dictionary lookup
  • ...and much more!

Disclaimers and privacy statement

  • Kitsuchan is not finished yet and may have bugs; stay tuned for new features and changes.
  • Kitsuchan may automatically leave bot collection servers.
  • Kitsuchan may do some anonymous logging information for debugging purposes. By using her, you implicitly agree to this logging. No personally-identifying information will be collected. If any personally-identifying information is collected for whatever reason, it will not be distributed to anyone else.
