RemindMeBot

Inviter

Préfixe: r> or @RemindMeBot

A bot designed for you to elegantly create, edit & be served reminders.

RemindMeBot does exactly what it says on the tin; a reminder bot. As a specialized bot, you can be assured RemindMeBot does a top notch job. With features like allowing you to set reminders using whichever method suits you best, viewing current reminders and forgetting them, custom prefixes per guild and (soon) very much more, RemindMeBot is unparalleled in his field.

Command Usage Info Example
help r>help or @RemindMeBot help Returns a help message to get you started with RemindMeBot. r>help
remindme r>remindme [<time argument> "<message>"] Creates a reminder. Pass without args to start a guided tour. r>remindme 12 hours "Buy groceries"
list r>list Sends you your current reminders in DM. Note: will send in channel if DMs are disabled. r>list
clear r>clear Starts a guided tour to clear all of your reminders. r>clear
forget r>forget Starts a guided tour to forget one of your reminders. r>forget
prefix r>prefix <desired prefix> Changes your prefix to the given argument. r>prefix !!
invite r>invite Returns an invite for RemindMeBot. r>invite
ping r>ping Returns the Websocket ping to the API servers in ms. r>ping
stats r>stats Returns information and statistics about RemindMeBot. r>stats
