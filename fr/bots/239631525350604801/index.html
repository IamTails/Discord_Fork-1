Pancake

Inviter

Préfixe: p! or @Pancake (customizable)

Simple to use multi-purpose Discord bot - music, moderation, fun and more!

The default prefix is p!.
Use p!help to get a list of commands. The prefix can be changed by using the "prefix" command.

Features:

Hosted 24/7. 99.9% uptime.

  • Simple Moderation
    • Kick, softban, ban & mute
    • Purge messages
    • Mute perist
  • Music System
    • Supports Soundcloud, YouTube (search/url/playlists) and Twitch
    • Pause & Resume
    • Music Queue
  • Information
    • User & Server Information Commands
    • Greeting & Farewell settings
  • Random Cat & Dog Images
  • Image generation
  • Advanced permission system
  • Autoroles
  • Mass role management
Add Pancake to your server and say p!help to get a list of commands.
