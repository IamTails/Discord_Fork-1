API Reference

Discord Fork is built on top of GitHub using GitHub pages and Jekyll, which means it is limited by GET requests only. Please interface with the GitHub API in order to automate actions with the bot list.

GET /api/bots/all.json
Obtain information on all bots. Some keys may be missing, such as the github key.
